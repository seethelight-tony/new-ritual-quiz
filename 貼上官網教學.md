# 怎麼把測驗貼到官網（BVSHOP HTML 元件）

> **目前採用：直接嵌入版（一鍵加入購物車）** ← 體驗最好的版本
> 2026-07-14 Tony 在後台開了「訪客結帳」權限後，BVSHOP 不再對訪客塞會員登入表單，所以能用這個版本。

## 你需要的檔案

`官網嵌入碼.html` —— 這份就是要貼的全部內容（已經切乾淨，不用自己找）。

## 步驟

### 1. 複製程式碼
打開：<https://github.com/seethelight-tony/new-ritual-quiz/blob/main/官網嵌入碼.html>
右上角按 **Copy raw file** 整份複製。

### 2. 到 BVSHOP 後台
- 編輯 `/one/new-ritual-quiz` 這一頁
- **先把舊的 HTML 元件內容整個清空**，再貼上新的 → 儲存

### 3. 用「全新無痕視窗」測試
- 一定要開**新的無痕視窗**（舊視窗 cookie 會殘留舊狀態，測不準）
- 玩一輪：選答案 → 看結果 → 按「🛒 一鍵加入購物車」
- 應該：**直接加進購物車、不跳登入**、按鈕變「去看看商品介紹 →」

## 客人的動線

```
玩測驗 → 看到專屬組合＋折扣碼 → 一鍵加入購物車（不用登入）
      → 「去看看商品介紹」認識商品、繼續逛
      → 想買時自己點右上角購物車 →「開始結帳」（訪客可直接買）
      → 想拿購物金的人，自己會選「立即登入」加入會員
```

## 上線前一定要做

到 BVSHOP 後台**建立折扣碼 `RITUAL30`**（現折 $30、設定「滿 $790 才可用」）。
不然客人看到折扣碼卻用不了，會客訴。

## 重要：不要關掉「訪客結帳」

這頁能正常運作的前提是後台的**「訪客結帳」權限保持開啟**。
若哪天關掉，BVSHOP 會在「訪客購物車有商品」時，自動在這頁塞一塊會員登入表單（新客會被嚇到）。
那種情況下就要改用備援方案 → `官網嵌入碼_iframe.html`（iframe 隔離版，代價是沒有一鍵加購、改成引導去商品頁選購）。

## 兩個版本怎麼選

| 版本 | 檔案 | 一鍵加購 | 何時用 |
|---|---|---|---|
| **直接嵌入**（現用） | `官網嵌入碼.html` | ✅ 有 | 訪客結帳開著時 |
| iframe 隔離（備援） | `官網嵌入碼_iframe.html` | ❌ 改為去商品頁 | 若關掉訪客結帳 |

## 改東西怎麼辦

折扣碼、商品、文案要改 → 跟 Claude 說，我改好 `index.html` 後重新產生 `官網嵌入碼.html`，你再重貼一次即可。
