# 怎麼把測驗貼到官網（BVSHOP HTML 元件）

## 你需要的檔案

`官網嵌入碼.html` —— 這份就是要貼的全部內容（已經幫你切乾淨，不用自己找）。

## 步驟

### 1. 複製程式碼
- 打開 GitHub 上的這份檔案：<https://github.com/seethelight-tony/new-ritual-quiz/blob/main/官網嵌入碼.html>
- 右上角有個「Copy raw file」按鈕（兩張紙重疊的圖示），按下去就整份複製好了
- （或用電腦打開 `C:\Users\Tony\new-ritual-quiz\官網嵌入碼.html`，Ctrl+A 全選 → Ctrl+C）

### 2. 到 BVSHOP 後台
- 進入你要放測驗的那一頁（一頁式頁面）的編輯畫面
- 新增一個 **「HTML 元件」**（就是那個會跳 JS/CSS 衝突警告的自訂元件）

### 3. 貼上
- 在 HTML 元件的輸入框裡 **Ctrl+V** 貼上
- 儲存 → 預覽整頁

### 4. 檢查
- 測驗應該會出現在你放的位置，寬度自動置中（最寬 430px，手機剛好滿版）
- 玩一輪：選答案 → 看結果 → 按「一鍵加入購物車」→ 應該真的加進購物車、按鈕變「前往結帳」

## 上線前一定要做的一件事

到 BVSHOP 後台**建立折扣碼 `RITUAL50`**（現折 $50、設定「滿 $790 才可用」）。
不然客人複製了折扣碼卻用不了，會客訴。

## 萬一貼上後沒顯示

有些頁面編輯器會把 `<script>` 擋掉。如果貼上後那塊是空白的，跟 Claude 說一聲「測驗貼上後沒出現」，我改用另一種嵌入方式（iframe）給你，只是 iframe 版的「加入購物車」會變成「跳到商品頁」。

## 改東西怎麼辦

折扣碼、商品、文案要改 → 跟 Claude 說，我改好 `index.html` 重新產生 `官網嵌入碼.html`，你再重貼一次即可。
