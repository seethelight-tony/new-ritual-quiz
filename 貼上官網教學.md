# 怎麼把測驗貼到官網（BVSHOP HTML 元件）

> **目前採用：iframe 版（軟性購物）**
> 原因：直接貼程式碼的版本，會讓 BVSHOP 在「訪客購物車有東西」時，自動在該頁塞一塊會員登入表單（嚇到新客）。改用 iframe 把測驗隔離，測驗不再碰購物車，就不會觸發登入表單。

## 你需要的檔案

`官網嵌入碼_iframe.html` —— 這份就是要貼的全部內容（很短，就一個 iframe＋一小段自動高度的程式）。

## 步驟

### 1. 複製程式碼
打開：<https://github.com/seethelight-tony/new-ritual-quiz/blob/main/官網嵌入碼_iframe.html>
右上角按 **Copy raw file** 整份複製。

### 2. 到 BVSHOP 後台
- 編輯 `/one/new-ritual-quiz` 這一頁
- **先把舊的 HTML 元件內容整個清空**（那是會觸發登入表單的舊版）
- 貼上新的 iframe 嵌入碼 → 儲存

### 3. 用「全新無痕視窗」測試
- 一定要開**新的無痕視窗**（舊視窗的 cookie 會殘留舊狀態，測不準）
- 玩一輪：選答案 → 看結果 → 按「看看你的專屬組合 →」→ 應該跳到套組商品頁
- 確認**整頁沒有出現會員登入表單**

## 客人的動線（軟性購物）

```
玩測驗 → 看到專屬組合＋折扣碼 → 按「看看你的專屬組合」
      → 到商品頁（自己選品項、加購物車）→ 想買時結帳（官網會員機制處理登入）
```

折扣碼區保留在結果頁，客人過去商品頁前就知道有優惠。

## 上線前一定要做

到 BVSHOP 後台**建立折扣碼 `RITUAL30`**（現折 $30、設定「滿 $790 才可用」）。
不然客人看到折扣碼卻用不了，會客訴。

## 已知限制（可接受）

- **沒有「一鍵加入購物車」**：iframe 是獨立網頁，碰不到官網購物車，所以改成引導去商品頁選購。這是為了避開登入表單的取捨。
- **少數情況仍可能看到登入表單**：如果客人**先前已經加過商品在購物車**、才來玩測驗，BVSHOP 還是會在這頁塞登入表單。新客（空購物車）不會遇到。要完全根除只能開「訪客結帳」（Tony 決定不開）。
